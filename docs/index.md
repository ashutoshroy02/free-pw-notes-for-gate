# 🚀 Free PW Notes for GATE

Welcome to the ultimate resource hub for **GATE (Graduate Aptitude Test in Engineering)** preparation! This repository contains a curated collection of high-quality notes, study materials, and practice problems (DPPs) covering Computer Science, Data Science, and Mathematics.

---

## 🌟 Why this repository?

Preparing for GATE requires structured materials. These notes are organized by subject and chapter to help you focus on one topic at a time.

- **Comprehensive**: Covers all major GATE subjects.
- **Organized**: Easy-to-navigate folder structure.
- **Free**: Open-access for all learners.

## 🛠️ How to use

1.  **Browse**: Use the sidebar to find the subject you're interested in.
2.  **Read**: Open the Markdown files or download the PDFs for offline study.
3.  **Practice**: Check the `DPP` folders within chapters for practice problems.
