# 🤝 Contribution Guide

First off, thank you for considering contributing to **Free PW Notes for GATE**! It's people like you that make the learning community better.

## 🚀 How can I contribute?

### 1. Adding New Notes
If you have high-quality notes for a subject not yet covered:
- Fork the repository.
- Create a new folder for the subject.
- Add your PDF or Markdown files.
- Submit a Pull Request!

### 2. Fixing Errors
Notice a typo or a wrong formula?
- Open an Issue describing the error.
- Or fix it yourself and submit a PR.

### 3. Improving Documentation
Help us make these docs even better by improving the descriptions or organization.

## 📝 PR Guidelines
- Ensure your notes are clear and legible.
- Use descriptive filenames (e.g., `Topic_Name_Class_Notes.pdf`).
- Briefly describe your changes in the PR description.
