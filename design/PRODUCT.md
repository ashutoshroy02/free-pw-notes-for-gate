# Project: PW-Gate Pro Documentation Hub

## Purpose
Transform a basic repository of PDF notes into a high-end, interactive educational dashboard that inspires focus and makes studying delightful.

## Users
- **Alex (Power User)**: Needs fast navigation to specific subjects.
- **Jordan (Confused Student)**: Needs a clear path through the subjects.
- **Casey (Mobile User)**: Needs a thumb-friendly interface for quick lookups on the go.

## Brand Personality
1. **Premium**: Feels like a high-end educational platform, not a folder.
2. **Dynamic**: Interactive elements that respond to user intent.
3. **Focused**: Ruthless removal of visual noise to emphasize content.

## Anti-goals
- No generic grids.
- No "slop" patterns (identically repeated icons).
- No standard side-stripe cards.
